# Notion Trading Widgets

## Enable GitHub Pages

1. Go to Settings
2. Click Pages
3. Under Source:
   - Deploy from branch
   - Branch: main
   - Folder: / (root)
4. Save

Your site will publish to:

https://hungrybrownguy.github.io/Notion-Trading/

---

## Embed in Notion

In Notion:
1. Type /embed
2. Paste one of these URLs:

Company Profile:
https://hungrybrownguy.github.io/Notion-Trading/profile.html?tvwidgetsymbol=NYSE:GME

Fundamentals:
https://hungrybrownguy.github.io/Notion-Trading/fundamentals.html?tvwidgetsymbol=NYSE:GME

To change symbols:
Replace NYSE:GME with e.g.:

NASDAQ:NVDA
AMEX:SPY
BINANCE:BTCUSDT
